# xavier maso's website

  * powered by [expressjs](http://expressjs.com/)
  * running live at [xaviermaso.com](http://xaviermaso.com/)

## Development
```bash
# automatically bundle css on change
npm run watch-css
```

## Bundle the assets

```bash
# bundle css and js
npm run build

# bundle only css
npm run build-css
# bundle only js
npm run build-js
```
